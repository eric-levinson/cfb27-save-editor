# Archive

These directories preserve the experiments that led to CFB27 Lua Hook. They
are unsupported, excluded from packages and releases, and may not run from
their relocated paths. Git history remains the authoritative source for their
original layout.

- `legacy-save-editor/` — the browser/Electron save editor and save-schema
  experiments.
- `legacy-hooks/` — remote injection, request detours, and live-rating probes.
- `research-tools/` — one-off reverse-engineering scripts and superseded plans.

The supported product is the root native host plus `packages/sdk` and
`packages/cli`. Start with the root README and maintained documents under
`docs/`.
