# Legacy save editor

Status: **unsupported historical snapshot**.

This code proved that CFB27 Dynasty saves could be decoded with the available
franchise schema, that selected settings could be changed safely, and that an
Electron/browser workflow could expose recruiting and roster data. It is not
the product maintained by this repository now. Relative paths and package
commands may no longer work after archival.

The runnable original layout is preserved in Git history at the pre-restructure
main commit `7498ae9` and the work merged through PR #4. The maintained
replacement is the Lua host, Node SDK, and CLI documented in the root README,
`docs/lua-api.md`, and `docs/safety.md`.
